Dentro de views
    Crear una carpeta con el nombre de la vista, en minusculas
    y que coincida con el nombre del controlador correspondiente
    dentro de la carpeta que se creó:
        Crear un archivo con el nombre del método a llamar.
        Por defecto está index
        La extensión debe ser .phtml

Example:

home/index.phtml
contact/index.phtml
contact/email.phtml
